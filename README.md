# LLMRank

MBA project — AI Ventures class.

A tool that helps small businesses understand and improve how they are represented by AI assistants (ChatGPT, Claude, Gemini, etc.).

## What it does

1. **Extract** — scrapes the business's website to understand their products, services, and positioning
2. **Measure** — queries multiple LLMs to evaluate how visible and accurately described the business is
3. **Improve** — delivers actionable recommendations to boost AI and web visibility

## Project structure

```
frontend/   Next.js app (TypeScript + Tailwind)
```

## Running locally

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
